# Review what you remember

Traditional machine learning is a different computing paradigm that traditional software development.

Traditional development involves explicit instructions to a machine to get a desired output
`input -> code -> output`

In machine learning, a model is trained with data. The process can be thought of approximating a function. The goal is to make a function that can reliably make predictions.
`features -> model -> prediction`

The course starts with linear regression. Approximates a function in the form of `y = mx + b`.

Supervised learning involves labeled output as part of training. Unsupervised learning purely relies on the data and patterns.

> What's the difference between classification and supervised learning?
Supervised & Unsupervised learning sit on the *training* axis.
Classification and regression problems sit on the *inference* axis.

Regression: predict a number
Classification: predict a category

prediction = features * weights
cost = prediction - actual
